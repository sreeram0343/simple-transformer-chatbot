# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public issue.
2. Send an email to the project maintainer at `security@example.com` (replace with actual contact if available).
3. Provide a detailed description of the vulnerability, including steps to reproduce it and a potential fix if possible.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible. We appreciate your help in keeping this project secure!
